# **Flood Risk Prediction using Machine Learning**
An end-to-end Machine Learning project that predicts flood risk levels based on environmental factors like rainfall, temperature, soil moisture, and river level. Built using Python, trained with the Random Forest algorithm, and version-controlled with Git & GitHub.
